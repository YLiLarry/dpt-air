Seamless Syncing

  • Runs in the system status bar
  • Sync via Bluetooth / Wi-Fi

Dpt Air runs in the background. You can sync through Bluetooth or Wi-Fi. Syncing can also be scheduled for a certain time interval. Changes on local computer and DPT device are compared and conflicts are automatically resolved.

Automatic Backup

  • No file can be lost by mistake
  • Rollback to a certain version from any time

Before syncing, Dpt Air backs up your file using Git. An interface is provided for extracting backups from any point of time.
